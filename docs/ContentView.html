Class: ContentView

ContentView

new ContentView()

ContentView can help you and your users view Content It will render Content into an li by default, and will use the Mustache template in templates/Content.html. But you can also specify your own `template` function when constructing.
Parameters:
Name Type Description
opts.model Content A Content model to view
opts.template Function A custom template function to use instead of the default
opts.defaultAvatarUrl Function An avatar to use if there is not one specified for the Author of Content
