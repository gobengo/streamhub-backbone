FeedView

new FeedView(sources)

FeedView is the default View for Collections of Content It is your basic feed/list view. http://d.pr/i/UtM0 New Content will stream in as the first children of the .el
Parameters:
Name Type Description
opts.collection Collection A Collection of Content (see models/Collection)
opts.contentViewOptions Object Options to be passed to any Content Views this instantiates This is useful for passing custom templates for Content
sources Object An object to configure stuff on a per-source basis Supports `twitter` and `rss` sub objects with the same opts as this root level
