new Content(props)

Stuff a user might want to look at. Content may be sourced from another Collection that lives in StreamHub's Cloud. When this is the case, Content is instantiated with json like this https://gist.github.com/4527966 Otherwise, it can be passed a string of bodyHtml (WIP)

props Object Properties that should be .set on the Content Model. Backbone style.

Name Type Description

bodyHtml String An HTML snippet for the main textual part of the Content
authorId String The ID of the Author in StreamHub
author Object Hydrated author data, if you have it handy
attachments oEmbed[] Attached media to display with the Content
createdAt Number Unix timestamp of when the Content was originally created
updatedAt Number Unix timestamp of when the Content was updated (perhaps edited)
